To upload file from console: 

            cat <file> | curl -F 'content=<-' http://{{ request.host }}
It's possible to set more properties: 
            cat <file.json> | curl -F 'timeout=6000' -F 'lang=json' -F 'one_use=on' -F 'content=<-' http://{{ request.host }}
